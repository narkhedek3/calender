# CALENDER

### HOW TO USE
> using npx: 
```properties
>npx calender-cli -h
npx: installed 28 in 6.137s
|************* CALENDER *************|
| -y => to specify year                    |
| -m => to specify month in numbers [1-12] |

-y -m options are mandatory to print calender

>npx calender-cli -y 2020 -m 8
npx: installed 28 in 3.963s
```
